# 📘 Assignment: Reading and Writing Data Files

## 🎯 Objective

Learn to work with files in Python by reading data, processing it line-by-line, and writing results to new files. These are essential skills for any data workflow and will prepare you for more advanced data analysis tasks.

## 📝 Tasks

### 🛠️ Read and Display File Contents

#### Description
Open a text file and display its contents. Learn how to safely open files, read data, and handle file paths in Python.

#### Requirements
Completed program should:

- Open a file using the `open()` function with proper context management (`with` statement).
- Read the entire file or specific lines from the file.
- Display the file contents in a readable format.
- Close the file automatically using the `with` statement.

### 🛠️ Process Data Line-by-Line

#### Description
Read a file containing structured data (for example: names, numbers, or comma-separated values) and process each line to extract or transform information.

#### Requirements
Completed program should:

- Read a file line-by-line.
- Parse each line to extract individual data points (for example: split by spaces or commas).
- Perform a transformation or calculation on the data (for example: convert to uppercase, calculate totals, filter by condition).
- Store processed results in a data structure (for example: list or dictionary).

### 🛠️ Write Results to a New File

#### Description
Take processed data and write it to a new file in a structured format. Use proper formatting to ensure the output is readable and useful.

#### Requirements
Completed program should:

- Create or open a new file for writing.
- Write processed data to the file in a clear format (one item per line or comma-separated values).
- Add headers or labels to make the output self-explanatory.
- Verify the file was created and contains the expected data.
